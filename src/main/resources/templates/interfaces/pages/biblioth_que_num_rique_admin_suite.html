Chapitre 3 : Les Équations du Second Degré

Soit l'équation ax² + bx + c = 0, où a, b et c sont des nombres réels avec a ≠ 0.

Le discriminant de cette équation, noté Δ (delta), est défini par la formule :

Δ = b² - 4ac

Trois cas se présentent selon le signe de Δ :

  • Si Δ > 0, l'équation admet deux solutions réelles distinctes.
  • Si Δ = 0, l'équation admet une solntion double.
  • Si Δ < 0, l'équation n'admet aucune solution dans ℝ.
